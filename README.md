# Aaryamann Goenka Portfolio

## Project overview

This repo hosts the portfolio site for Aaryamann Goenka, featuring robotics, AI, IoT, and cybersecurity projects built with Vite, React, and shadcn-ui/Tailwind styling.

## Getting started

1. Clone the repository (`git clone <YOUR_REPO_URL>`).
2. Install dependencies with `npm install`.
3. Run `npm run dev` to start the Vite dev server.
4. Visit `http://localhost:5173` (or the port reported by Vite) to preview the site.

## Building for production

- Run `npm run build` to produce a production bundle in `dist/`.
- Use `npm run preview` to locally preview the production build.

## Tools and tech

- Vite
- TypeScript
- React
- shadcn-ui / Radix components
- Tailwind CSS
